# Task 1: Digital Literacy Awareness Infographic

## Objective
To create a one-page infographic that helps batchmates 
understand digital literacy and safe internet practices.

## Tool Used
Canva (canva.com)

## Topics Covered
- What Digital Literacy means
- Safe Internet Practices
- Professional Online Presence

## Description
I designed this infographic using Canva to visually 
communicate key digital literacy concepts in a simple, 
student-friendly format. The design uses icons, colour 
coding, and short bullet points for easy readability.
