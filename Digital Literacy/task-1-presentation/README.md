# Task 1: Digital Literacy Awareness Infographic

## Objective

The objective of this task was to create an infographic that spreads awareness about digital literacy and safe internet practices.

## Description

In this task, an infographic was created using Canva to educate users about digital literacy, cybersecurity awareness, and responsible digital behavior. The infographic includes key topics such as safe internet usage, password security, email etiquette, and maintaining a professional online presence.

The infographic was designed to be visually engaging and easy to understand so that it can be used to spread awareness among students and internet users.
